Project ini memvisualisasikan puluhan kurva warna-warni yang menari, memantul, dan meninggalkan jejak (*trails*) estetis di layar. Setiap kurva memiliki "kepribadian" unikâ€”warna, kecepatan, ketebalan, dan lengkungan yang dihasilkan secara acak (prosedural).

## âœ¨ Fitur Utama

- **Visualisasi Dinamis**: Kurva bergerak bebas dengan fisika pantulan sederhana saat menyentuh batas layar.
- **Efek Jejak (Trails)**: Menggunakan teknik penumpukan layer semi-transparan untuk menciptakan efek jejak gerakan yang halus dan artistik.
- **6 Mode Variasi**: Setiap kurva memiliki salah satu dari 6 pola lengkungan yang berbeda secara acak.
- **Kontrol Interaktif**: Pengguna dapat memanipulasi jumlah kurva secara *real-time* menggunakan keyboard dan mouse.

## ğŸ® Kontrol (Controls)

Anda dapat berinteraksi dengan visualisasi menggunakan tombol berikut:

| Input | Aksi |
| :---: | --- |
| **Klik Mouse** | Menambahkan kurva baru tepat di posisi kursor mouse |
| **`A`** | Menambah 1 kurva acak |
| **`+`** | Menambah 5 kurva sekaligus |
| **`D`** | Menghapus 1 kurva terakhir (*Delete*) |
| **`-`** | Menghapus 5 kurva sekaligus |
| **`C`** | Menghapus semua kurva, membersihkan layar (*Clear*) |
| **`R`** | Reset ulang simulasi ke kondisi awal (79 kurva) |
